# Vazirmatn font files

Place the following **woff2** files in this folder for the extension to work:

- **Vazirmatn-Regular.woff2** (font-weight: 400)
- **Vazirmatn-Bold.woff2** (font-weight: 700)

## Where to get them

1. **GitHub (official)**  
   https://github.com/rastikerdar/vazirmatn/releases  

   Download the release ZIP, then from the `fonts/webfonts/` (or similar) folder copy:
   - `Vazirmatn-Regular.woff2`
   - `Vazirmatn-Bold.woff2`

2. **Google Fonts**  
   https://fonts.google.com/specimen/Vazirmatn  

   Use “Download family” and extract the `.woff2` files. Rename or ensure the filenames match exactly:
   - `Vazirmatn-Regular.woff2`
   - `Vazirmatn-Bold.woff2`

File names must match exactly (including case) for the extension’s `inject.css` to load them.
